## TD-H3 nicFW Channel Editor v2.4.0

### Highlights

- **RepeaterBook proximity** — **GMRS** and **amateur** proximity searches list repeaters without opening each **details** page first. **PL** / **TSQ** are fetched **only when you tap Import selected**, for **selected rows only**—faster search and fewer requests (same behavior as RadioDroid).

### Install

Download **`app-release.apk`** from this release and install on a supported Android version. See **`AndroidNICFW_CH_EDITOR/README.md`** and **`CHANGELOG.md`** in the repository for full feature context.
